# 🛒 MY Awesome Cart

**MY Awesome Cart** is a full-featured, modern e-commerce web application built using the Django framework. It provides a seamless shopping experience with a dynamic cart, order tracking, product search, a built-in blog, and secure user authentication.

---

## ✨ Features

* **Modern & Responsive UI:** Clean, card-based interface built with Bootstrap and custom CSS that looks great on both desktop and mobile.
* **Smart Shopping Cart:** JavaScript-powered cart using `localStorage` so users don't lose their items if they refresh the page.
* **Secure User Authentication:** Built-in Django authentication with sleek, pop-up modals for Login and Sign Up.
* **Order Tracking System:** Users can easily check the real-time status of their orders using their Order ID and Email.
* **Dynamic Search:** Quick and efficient product searching based on product names, categories, and descriptions.
* **Integrated Blog:** A dedicated space for articles, updates, and content marketing.

---

## 🛠️ Tech Stack

* **Backend:** Python, Django (v4.2)
* **Frontend:** HTML5, CSS3, JavaScript, jQuery
* **Styling Framework:** Bootstrap 4
* **Database:** SQLite (Default Django DB)
* **Image Handling:** Pillow
